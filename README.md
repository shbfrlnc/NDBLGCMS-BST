## Intro

NDBLGCMS-BST is a Node.js Blogging CMS.

Extract the zip source, cd to the folder, then (in command line):

```
npm install
```

Rename .env-example to .env.

Fill the data correctly, refer to .env-example.

Then:

```
npm start
```

Or

```
npm run dev
```

Default admin login:

```
username: admin@example.com
password: admin
```
